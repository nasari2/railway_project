Railway Management System (Django Project)

This is a Railway Management System built with Django and Python, designed to manage railway operations efficiently. The project implements functionalities for admin and worker roles, including task assignments, leave management, and worker registration. It provides a responsive web-based dashboard with statistics and real-time management tools for smooth workflow tracking.

Key Features:

Worker Registration: Add workers with details like username, email, phone, address, and job role.
Task Management: Assign tasks to workers, track completed and pending tasks.
Leave Management: Apply for leave, and admins can approve or reject leave requests.
Dashboard: Admin dashboard displays total workers, tasks, pending tasks, completed tasks, total leaves, and pending leaves.
Authentication: Secure login system with role-based access (Admin and Worker).
Responsive UI: User-friendly interface using HTML, CSS, and Bootstrap.

Tech Stack:

Backend: Django, Python
Frontend: HTML, CSS, Bootstrap
Database: SQLite (can be switched to MySQL)

This project is ideal for demonstrating CRUD operations, role-based access control, and dashboard analytics in a Django web application.
